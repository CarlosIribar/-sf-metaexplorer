# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-24

### Added

- Initial public release of `sf-metaexplorer`.
- Interactive TUI command `sf metadata explorer` for browsing Salesforce metadata.
- Compare, retrieve, and deploy workflows for supported metadata types.
- Org selection, sync cache, type subscriptions, and commit-preload support.
